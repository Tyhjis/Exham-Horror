Team Exham Horror: Unity project readme.

Run the game by double-clicking the .exe-file.


FPS-style survival horror game: 

-You must try to find 3 gramophones in order to survive!
-Watch out for the Shadow! Beware its deadly scream!
-You got only
	+ Gun with 40 bullets, and aint getting more.
	+ Flashlight which is mostly useless
	+ Your wits and gamer instincts


Known bugs:

-There is no actual credits scene, game just restarts when you get the 3 gramofones.
-Flashlight is weird downstairs.
